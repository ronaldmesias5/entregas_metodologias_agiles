## Resumen del Proyecto

**CALZADO J&R** es una plataforma web integral desarrollada bajo metodología ágil Scrum con el objetivo de digitalizar y optimizar la gestión de una fábrica de calzado. El proyecto se enfoca en crear un sistema robusto de registro e identificación de usuarios con autenticación segura basada en JWT tokens, gestión granular de roles y permisos, y validación rigurosa de datos. La plataforma incluye un backend escalable construido con FastAPI y Python, un frontend moderno y responsivo desarrollado en React, y una base de datos relacional PostgreSQL. El sistema fue diseñado siguiendo estándares de seguridad actuales (bcrypt para contraseñas, JWT para sesiones) y arquitectura de microservicios containerizada con Docker. El desarrollo se realiza de forma incremental a través de sprints de dos semanas, entregando funcionalidades completamente testeadas en cada iteración.

## Manejo de Sprints
