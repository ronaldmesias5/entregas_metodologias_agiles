## Resumen del Proyecto

**CALZADO J&R** es una plataforma web integral desarrollada bajo metodología ágil Scrum implementada con el objetivo de digitalizar y optimizar la gestión y producción de la fabrica. El proyecto se pretende que sea un sistema robusto de registro e identificación de usuarios con autenticación segura basada en JWT tokens, gestión de roles y permisos, y validación rigurosa de datos. La plataforma pretende incluir un backend escalable construido con FastAPI y Python, un frontend moderno y responsivo desarrollado en React, y una base de datos relacional PostgreSQL. El sistema será diseñado siguiendo estándares de seguridad actuales (bcrypt para contraseñas, JWT para sesiones) y arquitectura de microservicios containerizada con Docker. El desarrollo se realizará de forma incremental a través de sprints de dos semanas, entregando funcionalidades completamente testeadas en cada iteración.

## Manejo de Sprints

- **Duración:** 2 semanas (15 días)
- **Historias por Sprint:** 2 historias de usuario
- **Metodología:** Scrum
- **Total de Sprints:** 10
